## 🔬 实验建议

完成学习后，建议做以下实验来深化理解：

### 实验 1：验证你的部署（1 小时）

```bash
# 对 5 个正在运行的部署：
1. 用公式估算显存需求
2. 用 nvidia-smi 查看实际占用
3. 计算"显存利用率" = 实际占用 / 容量
4. 如果实际 > 估算：可能用了优化或量化
5. 记录成一个表格（做为学习总结）
```

### 实验 2：计算器练习（30 分钟）

```bash
# 运行 llm_memory_calculator.py
python3 llm_memory_calculator.py --model qwen2.5-14b
python3 llm_memory_calculator.py --model qwen2.5-7b --precision int8
python3 llm_memory_calculator.py --param-billion 70 --batch-size 4
python3 llm_memory_calculator.py --param-billion 70 --seq-len 4000

# 理解每个参数的效果
```

### 实验 3：显存利用率优化（2+ 小时）

```
目标：找出你们部署中"最不经济"的配置

步骤：
1. 列出所有部署（模型 + GPU）
2. 用公式计算理论显存需求
3. 计算显存利用率
4. 找出显存利用率 < 40% 的部署
5. 提出优化建议（量化/换模型/混合部署）

这会让你真正理解"显存经济学"
```
