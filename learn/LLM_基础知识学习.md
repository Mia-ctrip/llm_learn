# 0. LLM
LLM 是一个具有固定层数结构的深度神经网络（通常由几十层 Transformer Block 堆叠而成）。每一层包含大量的权重（weights）和偏置（biases），这些参数在训练阶段通过反向传播不断调整，目的是最小化损失函数——即让模型在给定输入序列后，预测的下一个 token 的概率分布尽可能接近真实的下一个 token。训练完成后，权重被固定下来，模型进入推理阶段，用于实际生成文本。
# 1. 隐藏层（层的位置）：
输入层和输出层之间的所有层都叫隐藏层。输入层并非神经网络层，没有神经元，输出层是神经网络层。
隐藏层可以有多层，层数由模型结构（超参数）决定。每一层由若干神经元组成，每个神经元都有自己的权重 W 和偏置 b，并通过一个预先选定的激活函数进行非线性变换。
训练的目的，是通过反向传播和梯度下降不断调整所有层的 W 和 b，使得训练集的输入经过这些隐藏层的逐层变换后，最终的输出尽可能贴近真实标签（即最小化损失函数）。
# 2. 全连接层（层的连接方式）：
全连接层是一种"维度变换 + 信息融合"的通用层：它让每个输出神经元都能访问所有输入神经元，通过可学习的权重矩阵 W，把输入向量投影到任意目标维度——既可以降维（压缩信息），也可以升维（扩展表达空间），还可以保持维度不变（纯粹变换）。
训练的本质，是让这个矩阵 W 学会"如何有选择地组合输入信息"，而不是"保留所有信息"。信息在通过全连接层时会被重新编码，有用的特征被放大，无关的特征被抑制。
卷积层、池化层、循环层、注意力层、Dropout层都是非全连接层。
# 3. 激活函数：
激活函数是神经网络中预先选定的非线性变换函数，它通过给每个神经元的线性输出（W·x+b）套上非线性壳，使得多层网络得以拟合复杂的非线性关系。其选择不是随意的，而是由"所在层的功能"和"任务类型"共同决定——隐藏层重在引入非线性（首选 ReLU 系列），输出层重在匹配任务输出的含义（分类用 Softmax/Sigmoid，回归则省略）
# 4. 前向传播
前向传播是指：输入值从输入层开始，逐层通过每一层的计算——先做线性变换 Wx+b，再套非线性激活函数 f(·)——一路传递到输出层得到预测值 ŷ；然后通过损失函数计算预测值 ŷ 和真实值 y 之间的损失 Loss。整个过程是纯数值计算，最终产出一个具体的损失数值，作为反向传播的起点。
损失函数有损失函数分为用于预测连续值的分类损失（均方误差等）和用于预测分类的回归损失（交叉熵损失）两类，LLM 用的就是 Cross-Entropy。
一次完整的前向传播会依次经过神经网络的每一层，每层的所有神经元各自做 f(W·x+b) 的计算， Loss是一个关于所有参数 (W₁, b₁, W₂, b₂, …) 的复合函数。
# 5. 神经网络训练
神经网络训练的本质，是在参数空间中求解一个优化问题——在模型结构（层数、每层神经元数、激活函数等）固定的前提下，找到让损失函数 `Loss(W, b)` 尽可能小的一组权重 W 和偏置 b。从数学视角看，训练数据 (x, y) 是已知的，**W 和 b 才是真正的"自变量"**，Loss 是它们的函数。
虽然理论上多变量函数的极值在"梯度=0"的地方，但在深度学习中无法走"解方程"这条路，原因有三：
1. **参数量太庞大**（动辄千亿级），非线性方程组根本无法解析求解；
2. **Loss 函数极其非凸**，梯度=0 的点有无数个（局部最小、鞍点、全局最小），不都是想要的；
3. **Loss = 0 也不是好的训练结果**——可能过拟合，而且实际上由于数据噪声和理论下界，也根本达不到。
因此实际采用的是迭代优化的思路——**梯度下降**：从随机初始化的参数出发，每一步通过前向传播算出当前 Loss、反向传播算出梯度，然后朝**当前位置梯度的反方向**走一小步，如此循环几十万甚至几百万次，直到 Loss 收敛到一个"足够好"的值。
**训练的目标**是让 Loss 最小，**梯度下降是实现这个目标的手段**——不是"找某组特殊参数"，而是"走一条下降的路径"，每一步的方向都由当前位置的梯度动态决定，最终的参数是**迭代走出来的结果**。
> 训练不是"求解"最优参数，而是"搜索"最优参数；不是"一步到位"，而是"千万次小步"；不是"全局最优"，而是"足够好的局部最优"。**这是深度学习从"数学理论"走向"工程实践"的关键思维转换。**
一次完整的训练流程有以下三个步骤：
* 前向传播 数据 → 网络 → 预测值 ŷ → Loss
* 反向传播 Loss → 梯度 ∇Loss(每个参数的偏导) 链式法则
* 更新参数  梯度 → 更新参数(SGD/Momentum/AdaGrad/Adam) 
# 6. 损失和损失函数
损失函数是在前向传播的最后一步计算的数学公式，用来量化神经网络预测值 ŷ 和真实值 y 之间的差距。它产出一个具体的数值——损失值 Loss。
Loss = (ŷ - y)²  =  (W·x + b - y)² / Loss(W, b) 损失函数是关于w b的函数,是权重参数的函数
* 作为评估指标：衡量当前参数下模型的好坏，一般情况下越小越好
* 作为反向传播的起点：从 Loss 出发，沿路径反向计算每个参数的梯度
按任务类型选择不同的损失函数：
回归任务 → 均方误差（MSE）
分类任务 / LLM → 交叉熵（Cross-Entropy）
# 7. 梯度和梯度下降
梯度是损失函数Loss(W, b)对每个参数（权重 W 和偏置 b）求偏导得到的向量。(梯度的变量是 W₁, W₂, …, b₁, b₂, …（所有可训练参数）——不是输入 x。)
梯度是关于可学习参数的变量，可学习参数每次迭代超梯度反方向移动能保证梯度始终下降。
梯度下降算法是每一轮训练中通过调整参数来让损失值朝更小方向变化的算法。调整方法是：让每个参数减去 它对应的偏导数乘以学习率 —— 也就是朝着"偏导数的反方向"移动一小步。
```
W_new = W_old - η × (∂Loss/∂W)
              ↑       ↑
           学习率   该参数的偏导数
              └───────┬───────┘
                      │
               移动的"量"（有方向、有大小）
```               
# 8. 学习率
学习率（Learning Rate, η）是控制梯度下降中每次参数更新"步长"大小的一个乘数。具体来说，每个参数每次的移动量等于：

移动量 = η × 该参数的偏导数
学习率是超参数（Hyperparameter）——由人为设定，不通过训练学习得到。它的取值通常是一个远小于 1 的小数，典型值在 0.00001 ~ 0.01 之间，模型规模越大，学习率通常越小。
* 太小 → 每步更新幅度小，训练收敛非常慢（浪费算力和时间，但通常不会训练失败）
* 太大 → 在最优点附近来回震荡，甚至 Loss 爆炸（NaN），训练失败

# 9. 链式法则
链式法则是微积分中用于求复合函数导数的基本规则，它是反向传播算法的数学基础。在神经网络中，Loss 是由多层嵌套计算构成的复合函数，链式法则让我们可以逐层拆解，高效地计算每个参数的梯度。
**基本形式：** 如果 `y = f(u)` 且 `u = g(x)`，则：
```
dy/dx = (dy/du) × (du/dx)
       ↑         ↑
    后一步    前一步
```
**神经网络中的应用：** 对于 `Loss = f(z)` 且 `z = W·x + b`：
```
∂Loss/∂W = (∂Loss/∂z) × (∂z/∂W)
           ↑              ↑
        后续计算      当前层计算
```
**矩阵运算中的链式法则：** 神经网络中的运算是矩阵乘法，不是标量乘法。对于 `z = W·x + b`：
* **梯度往前传**（求 ∂Loss/∂x）：`∂Loss/∂x = W^T · (∂Loss/∂z)` — 需要**权重转置**才能维度匹配
* **权重梯度**（求 ∂Loss/∂W）：`∂Loss/∂W = (∂Loss/∂z) · x^T` — 用**输入转置**做外积，结果形状和 W 一致
* **偏置梯度**（求 ∂Loss/∂b）：`∂Loss/∂b = ∂Loss/∂z` — 直接等于（多样本时需要按样本维度求和）

**记忆技巧：** 往**前**传梯度用 **W^T**，算**权重**梯度用 **x^T**（前一层激活值的转置）。这些转置操作不是随意的，而是矩阵微积分的必然结果，保证了梯度的形状和对应参数一致。

# 10. 反向传播
反向传播（Backpropagation）是训练神经网络的核心算法，它通过链式法则高效计算 Loss 对所有参数的梯度。**反向传播只负责计算梯度，不负责更新参数**——参数更新由优化器完成，这是两个独立的步骤。

**为什么需要反向传播：** 前向传播只能得到 Loss 的数值（比如 0.235），这只告诉我们"模型有多差"，但不告诉我们"该怎么调参数"。Loss 的数值本身无法直接用来调参数，因为：
* 每个参数可以往任意方向调整（增大或减小）
* 不知道调整哪个参数对 Loss 影响更大
* 不知道调整的幅度该是多少

**梯度才是答案：** 梯度 `∂Loss/∂W` 告诉我们"W 增加一点点，Loss 会如何变化"：
* 如果 `∂Loss/∂W = 0.08`（正数）→ W 增加会让 Loss 增加 → 应该**减小** W
* 如果 `∂Loss/∂W = -0.03`（负数）→ W 增加会让 Loss 减少 → 应该**增大** W
* 统一写成：`W_new = W_old - learning_rate × ∂Loss/∂W`（朝梯度反方向走）

**计算流程：** 反向传播分为两个阶段，从输出层开始逐层往前计算。

**阶段1：逐层计算误差项 δ（从后往前传递梯度）**
```
# 输出层（反向传播起点）
δₙ = (∂Loss/∂output) ⊙ σ'(output)
   = (output - y) ⊙ output(1 - output)  # MSE + Sigmoid

# 隐藏层（链式法则递推）
δᵢ = (δᵢ₊₁ · Wᵢ₊₁^T) ⊙ σ'(aᵢ)
     ↑               ↑
  后一层误差     当前层激活函数导数
```

**阶段2：用误差项计算每层参数的梯度**
```
∂Loss/∂Wᵢ = aᵢ₋₁^T · δᵢ    # 前一层激活值转置 × 当前层误差项
∂Loss/∂bᵢ = sum(δᵢ)        # 按样本维度求和
```

**完整例子（2层网络）：**
```
前向传播：
  z1 = W1·x + b1  →  a1 = sigmoid(z1)
  z2 = W2·a1 + b2  →  a2 = sigmoid(z2)
  Loss = 0.5(a2 - y)²

反向传播：
  # 输出层
  δ2 = (a2 - y) ⊙ a2(1-a2)
  ∂Loss/∂W2 = a1^T · δ2
  ∂Loss/∂b2 = δ2
  
  # 隐藏层
  δ1 = (δ2 · W2^T) ⊙ a1(1-a1)
  ∂Loss/∂W1 = x^T · δ1
  ∂Loss/∂b1 = δ1
```

**形状验证技巧：** 梯度的形状必须和对应参数一致。如果 `W: (m, n)`，则 `∂Loss/∂W` 也必须是 `(m, n)`。通过转置操作保证维度匹配是矩阵运算的必然要求。

> 反向传播的本质是**用链式法则把复杂的多层嵌套求导，拆解成逐层的简单计算**。每层只需要知道"后一层的梯度"和"本层的激活函数导数"，就能算出"传给前一层的梯度"和"本层参数的梯度"。这种逐层递推的设计，使得无论网络有多少层，计算复杂度都是线性的——这是深度学习能够训练几十层甚至上百层网络的关键。

# 11. 优化器
优化器(Optimizer) 是训练中"用梯度更新参数"的算法,它是训练闭环中的独立一步,在反向传播之后执行,目的是让可学习参数朝着让Loss下降的方向更新。
结合了动量法(Momentum):加入历史梯度的"惯性"和自适应学习率法(AdaGrad / RMSprop):为每个参数调整专属步长的Adam是LLM的主流选择。
